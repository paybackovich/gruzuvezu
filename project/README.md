# Gruz Uvezu – локальный docker-проект

Минимальный каркас для SPA + API + Nginx-провайдера, который можно запустить локально через `docker-compose` и без изменений перенести на Timeweb (или любой другой сервер с Docker).

## Состав

- `frontend/` – Vite + React SPA с четырьмя страницами.
- `backend/` – Express-сервер с единственным эндпоинтом `GET /api/health`.
- `nginx/` – конфигурация, которая отдаёт статические файлы и проксирует `/api` на backend.
- `docker-compose.yml` – связывает билд фронтенда, backend и nginx.

## Запуск локально

```bash
cd project
docker-compose up --build
```

После успешного билда:

- Сайт: <http://localhost>
- Проверка backend через прокси Nginx: <http://localhost/api/health>

Команда `--build` гарантирует, что контейнер `frontend-build` соберёт свежий `/dist` и положит его в том, который использует Nginx.
После завершения сборки контейнер останавливается — при следующем запуске команды сборка выполнится заново.

## Как вносить изменения

1. Вносите правки в исходники (`frontend/src`, `backend/src`).
2. Повторно выполните `docker-compose up --build`, чтобы пересобрать образ backend и обновить статические файлы фронтенда.

## Подготовка к деплою на Timeweb

- Обновите `server_name` в `nginx/default.conf`, указав домен `paybockovich.site`.
- Убедитесь, что на сервере открыт порт 80 (и 443, если планируете TLS) и проброшен наружу в панели Timeweb.
- Скопируйте весь каталог `project` на сервер. На Timeweb выполните те же команды `docker-compose up --build -d`.
- (Опционально) добавьте TLS / редиректы непосредственно в `nginx/default.conf`.

На этом всё — структура уже готова к развёртыванию и дальнейшему развитию.
